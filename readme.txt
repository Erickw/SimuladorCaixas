Simulador de caixas de Supermercados - Erick Willy - 2015032256

---------------------------------------------------------------------------------------------------------------------------------------------------------------

O que foi feito:

	-> Classe para lista encadeada generica incluindo suas principais funcionalidades: Consulta de quantidade de elementos da lista, Consulta de elementos especificos inserção e remoção tanto no começo quanto no fim da lista.

	-> Construtores sobrecarregados para definir o comportamento da lista respeitando as operações de cada tipo de lista, podendo ser fila ou pilha.

	-> Classes especificas e seus atributos para o supermercado baseadas na situação real, como Clientes, Caixas, Supermercado, etc.

	-> O programa cria e adiciona caixas e clientes, insere, busca e remove o cliente em um fila de caixa, seleciona o caixa com a menor fila para inserir um novo cliente.

------------------------------------------------------------------------------------------------------------------------------------------------------------------


*COMO COMPILAR*

Basta extrair os arquivos simu_caixas.h simu_caixas.inl e simu_caixas.cpp na mesma basta, acessar a pasta através do comando "CD" do terminal do linux
e compilar através do comando: g++ simu_caixas.cpp -o simulador Em seguida, para executar, o comando a ser usado deve ser o: ./simulador.
